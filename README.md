# StitchCraft Tailoring & Stitching Studio

A premium, frontend-only website template built without any frameworks.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript

## Project Structure
- `index.html`: Master template for the homepage
- `pages/`: Additional placeholder pages (about, services, etc.)
- `css/`: Modular CSS files covering variables, global styles, components, animations, and responsiveness.
- `js/`: Modular JS files for theme logic, RTL toggling, mobile navigation, and components.
- `assets/`: Empty directories ready to receive images, fonts, and icons.

## Features
- **Light/Dark Theme Toggle:** Integrated and persists using `localStorage`.
- **RTL/LTR Toggle:** Integrated for accessibility and translation readiness, persisting via `localStorage`.
- **Responsive Navigation:** Hamburger menu designed for smaller screens.
- **Micro-Animations:** Clean loading states, transitions, and a Back-To-Top button.
- **Premium Aesthetics:** Uses rich color palettes and high-contrast styling fitting a luxury brand.

## How to Run
Since this is a vanilla frontend project, no build step or backend server is strictly required. Simply open `index.html` in any modern web browser or serve it using an extension like VSCode's Live Server.
